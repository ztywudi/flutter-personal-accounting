## 📱 获取APK的方式

### 方式1：等待GitHub Actions自动构建（推荐）
1. 访问：https://github.com/ztywudi/flutter-personal-accounting/actions
2. 点击最新的构建任务（Run #1）
3. 等待构建完成（通常需要5-10分钟）
4. 构建完成后，在"Artifacts"区域下载 `app-release.apk`

### 方式2：本地构建
如果你有Flutter开发环境：
```bash
# 克隆仓库
git clone https://github.com/ztywudi/flutter-personal-accounting.git
cd flutter-personal-accounting

# 安装依赖
flutter pub get

# 构建APK
flutter build apk --release

# APK位置
# build/app/outputs/flutter-apk/app-release.apk
```
