# 个人记账应用 - 构建指南

## 项目说明

这是一个使用Flutter开发的个人记账应用，包含基本的收支记录、分类管理和统计功能。

## 文件说明

- `flutter-personal-accounting.tar.gz` - 完整的Flutter项目源码包
- 解压后使用Flutter SDK构建Android APK

## 构建步骤

### 1. 准备环境

确保你的电脑已安装：
- Flutter SDK (建议版本 3.0.0 或更高)
- Android Studio (包含Android SDK)
- Git

### 2. 解压项目

```bash
tar -xzf flutter-personal-accounting.tar.gz
cd flutter-personal-accounting
```

### 3. 安装依赖

```bash
flutter pub get
```

### 4. 构建APK

**调试版（推荐先测试）：**
```bash
flutter build apk --debug
```
生成的APK位置：`build/app/outputs/flutter-apk/app-debug.apk`

**发布版（正式使用）：**
```bash
flutter build apk --release
```
生成的APK位置：`build/app/outputs/flutter-apk/app-release.apk`

### 5. 安装到手机

将APK文件传输到Android手机，在手机上允许安装未知来源应用，然后安装即可。

## 如果遇到问题

### 问题1：Flutter版本不兼容
**解决方案：** 项目使用Flutter 3.0.0开发，如果你使用更高版本，可能需要：
1. 修改 `pubspec.yaml` 中的SDK版本要求
2. 运行 `flutter pub get` 更新依赖

### 问题2：Android SDK找不到
**解决方案：**
1. 打开Android Studio
2. 安装Android SDK（Settings → Android SDK）
3. 设置环境变量 `ANDROID_SDK_ROOT` 指向SDK路径

### 问题3：依赖安装失败
**解决方案：**
1. 检查网络连接
2. 尝试使用国内镜像（修改 `flutter/pkg/flutter_tools/lib/src/dependency_checker.dart`）
3. 或者手动下载依赖包

## 功能说明

应用包含三个主要页面：
1. **首页** - 显示收支统计和记录列表
2. **添加记录** - 添加收入或支出记录
3. **统计** - 查看分类统计（待完善）

## 数据存储

应用使用SQLite数据库本地存储数据，文件位置：
- Android: `/data/data/com.example.flutter_personal_accounting/databases/`
- iOS: `Documents` 目录
