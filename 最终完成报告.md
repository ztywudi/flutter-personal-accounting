3. **下载APK**
   - 构建完成后，页面右侧会显示 "Artifacts" 区域
   - 点击 "app-release" 下载APK文件

### 方法2：本地构建

如果你有Flutter开发环境：

```bash
# 1. 克隆仓库
git clone https://github.com/ztywudi/flutter-personal-accounting.git
cd flutter-personal-accounting

# 2. 安装依赖
flutter pub get

# 3. 构建APK
flutter build apk --release

# 4. APK位置
# build/app/outputs/flutter-apk/app-release.apk
```
