如果你有电脑和Flutter环境：

```bash

git clone https://github.com/ztywudi/flutter-personal-account-ing.git
cd flutter-personal-account-ing

# 2. 安装依赖
flutter pub get

# 3. 构建APK

# 4. APK位置
```
